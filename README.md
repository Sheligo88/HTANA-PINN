# H-TANA PINN
Physics-Informed Neural Network for cosmological anisotropy analysis.

Analyzes the Pantheon+ supernova catalog using a free-form E(z) 
reconstruction with the Friedmann equation as a soft constraint.

**Main results:**
- Bulk flow: vp = 506 ± 251 km/s
- Kinematic decomposition: ε_int ≈ 0
- Tsallis index: Δ = 1.52 (persistent under ablation)
